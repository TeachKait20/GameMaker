# GameMaker

GameMaker — это среда разработки игр, предназначенная для создания 2D-игр (и частично 3D). <br><br>
В ней можно делать полноценные игры без глубоких знаний программирования, но при этом она поддерживает собственный язык — GML (GameMaker Language), который позволяет реализовать сложную логику.

Основные возможности:
* Редактор сцен и спрайтов — можно прямо в программе рисовать персонажей, объекты и анимации.
* Drag-and-Drop система — визуальное программирование: можно собирать механику без написания кода.
* Язык GML (GameMaker Language) — похож на Python или JavaScript, используется для расширенного управления логикой.
* Редактор звуков и шейдеров — встроенные инструменты для работы с аудио и визуальными эффектами.
* Экспорт на: Windows, macOS, Linux, Android, iOS, HTML5 и игровые консоли.

Скачать бесплатную версию (запуск/создание игр для Windows) можно с официального сайта: https://gamemaker.io/en
